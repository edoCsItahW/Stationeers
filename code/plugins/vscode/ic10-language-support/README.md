# IC10 Language Support

IC10 language support for Visual Studio Code.

## Features

- **Hover Information**: Hover over IC10 tokens to see detailed information
- **Syntax Highlighting**: Full syntax highlighting for IC10 assembly language
- **Diagnostics**: Real-time error detection and reporting

## Requirements

- VS Code 1.125.0 or higher

## Extension Settings

This extension contributes the following settings:

- `ic10.language`: Set the language for error messages (`zh-hans` or `en-us`)
